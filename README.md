# 🚀 ECS Python Flask App Deployment (Docker + AWS ECR + ECS Fargate)

## 📌 Project Overview

This project demonstrates a complete DevOps workflow for deploying a containerized Python web application to AWS using modern cloud-native tools.

A simple Flask application is built, containerized using Docker, pushed to Amazon ECR, and deployed on Amazon ECS using Fargate for serverless container orchestration.

---

## 🏗️ Architecture
GitHub → Docker → ECR → ECS Fargate → Browser
